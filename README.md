# 常春藤進階字彙 Level 5 語音撥放器

## 使用方法

1. 點擊以下連結前往播放器：  
    https://gallen881.github.io/IvyAdvancedVocabularyLevel5AudioPlayer/
2. 翻開**常春藤進階字彙 Level 5**第 9 頁，輸入 Unit 1 最後一個單字。
3. 輸入欲聆聽的音軌編號。  
    e.g. `0201-0207`
4. 使用撥放、下一頁等控制鈕調整。
